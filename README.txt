Projet 11 : Partage de compte

Sujet : Faire une application similaire à Tricount ou Splitwise. L’application gère une liste d’utilisateur et leurs dettes respectives, elle propose les sommes à transférer pouréquilibrer toutes les dettes. 

Par exemple : Alice, Bob et Chloé vont boire un verre Alice paye lapremière tournée 16€, Bob paye la seconde 20€. 
Pour équilibrer l’application, propose que Chloérembourse 4€à Alice et 8€à Bob.

Une version simple de l’application serait indépendante et fonctionnerait sur un seul téléphone,une version plus complexe pourrait se synchroniser entre des téléphones par wifi, Bluetooth ou unserveur web.

Indication pour ce projet :  

Pour ajouter un projet :
- Appuyer sur le menu en haut à gauche
- Choisir "New Projet"
- Remplir des informations
- Appuyer la button "COUNT" si vous avez fini
Note: Celui-là va enregitre des informations ce que vous avez rempli dans la base de donnée

Pour ajouter un participant :
- Remplir des informations
- Appuyer la button "ADD" si vous avez fini

Pour supprimer un participant :
- Appuyer un peu longue 
- Un dialog va afficher pour que vous pouvez choisir

Pour supprimer un projet :
- Appuyer sur la button "DELETE"

Pour voir la list de projet :
- Appuyer sur le menu en haut à gauche
- Choisir "List Projet"

Pour entrer un projet dans la liste projet :
- Appuyer sur l'image à droite dans la même sens de la projet que vous voulez entrer
